### Brave Search — keyword-поиск с мульти-страновой выдачей

Нативный tool OpenClaw (вызывается напрямую, НЕ через exec/mcporter):

```
web_search(query: "запрос", country: "us", count: 10)
```

Дополнительный к Exa. Лучше для: конкретных фактов/чисел, свежих новостей (<24ч), точных названий, региональных запросов.

Параметры: `query`, `count` (1-10), `country` (ISO код), `freshness` (`pd`/`pw`/`pm`/`py`).
Rate limits: 1 QPS, 2000 запросов/мес.

#### Мульти-страновой поиск

Для каждого запроса определи, в каких странах могут быть релевантные результаты, и сделай несколько запросов с разными `country`. Не ограничивайся одной страной.

Примеры:
- "купить подписку ChatGPT дёшево" → искать в `us`, `tr`, `ar`, `in`, `br` — цены разные в разных регионах
- "лучшие VPN 2026" → искать в `us`, `gb`, `de`, `ru` — обзоры разные в разных странах

Правило: если тема не привязана к одной стране — ищи минимум в 3-5 странах. Сам определи какие страны релевантны для конкретного запроса. Больше выдач = больше данных для анализа.
